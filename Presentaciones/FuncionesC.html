Deﬁnición

Una función es un fragmento de código que recibe cero, uno o más argumentos y que produce como resultado uno o varios valores. La función tiene una tarea u objetivo específico.

En MatLab las funciones predefinidas por el lenguaje incluyen, entre otras:

● sqrt(x) ● pow (x,y) ● fix(value) ● randi(x,y)

Uso

Regla: Si hay posibilidad de que alguien tuviera el mismo problema que tu para resolver, seguramente ya hay alguna función diseñada que lo resuelva.

Las funciones tiene como objetivo separar la solución a un problema en subsecciones, el código está mejor organizado y hay posibilidad de reusar código y hacer una solución eficaz.

Se vuelven una herramiento, tipo caja negra, a la cual puedes llamar cuando lo necesites

¿Cómo se deﬁne una función?

[tipo de regreso] nombre_función( argumentos) {

Funciones que no devuelven

Una función que no devuelve se considera un procedimiento y debe tener un encabezado como el que se muestra:

void funcionx(int a, double b) {

//no debe tener return dentro del cuerpo de la función

}
